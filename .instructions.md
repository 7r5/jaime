# Copilot Instructions

Este repositorio es un proyecto de bot de WhatsApp con backend en Python/FastAPI y frontend en React.

## Contexto principal

- El backend está en `backend/`.
- El frontend está en `frontend/`.
- El objetivo principal es mantener y mejorar el bot de WhatsApp y su integración con el webhook.
- El backend usa Poetry; el entorno virtual local puede crearse en `backend/.venv` si Poetry está configurado así, pero no debe versionarse ni subirse a Git.

## Reglas de edición

- No modificar configuraciones de despliegue a menos que el cambio sea necesario para el flujo local.
- No forzar un intérprete Python fijo en `.vscode/settings.json`; selecciona el intérprete de Poetry desde la paleta de comandos.
- Para cambios en el backend, preferir `backend/main.py`, `backend/agent.py`, `backend/whatsapp.py` y `backend/routes.py`.

## Sobre dependencias

- El backend usa `poetry` y el entorno virtual se administra con Poetry. No se incluye en el repositorio.
- El frontend usa `npm`.

## Prioridades

1. Mantener el bot funcionando localmente.
2. Preservar las variables de entorno usadas para WhatsApp y IA.
3. Garantizar que la configuración de VS Code sea específica por workspace.
